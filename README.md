# skill_tracker

A simple command-line based Skill Tracker built using Python to help manage and monitor learning progress.

This project allows users to add skills, update progress, and track completion status in a clean and interactive way.

## 🚀 Features

* Add new skills with progress tracking
* Update progress dynamically
* Mark skills as completed
* View all skills in a structured format
* Input validation for better user experience

## 🛠️ Tech Stack

* Python (Core concepts)
* Dictionaries for data storage
* Functions and loops for modular design

## 🧠 Concepts Used

* Data structures (dictionary-based storage)
* Conditional logic and loops
* User input handling and validation
* Modular function design

## ▶️ How to Run

```bash
python skill_tracker.py
```

## 📌 Example Workflow

* Add a skill (e.g., Python, Gym, Communication)
* Update progress (e.g., "Basics done", "Intermediate")
* Mark it as completed once finished
* View all tracked skills anytime

## 🔮 Future Improvements

* Save data to file (persistent storage)
* Add deadlines and reminders
* Create a GUI version
* Add progress percentage tracking
